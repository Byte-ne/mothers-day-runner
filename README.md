# Mother's Day Game

A simple, browser-based arcade game created as a personalized Mother's Day gift. The goal is to catch hearts while dodging stress objects to get the highest score.

### Technologies Used

*   **HTML5 Canvas**: For rendering the game entities and particles.
*   **CSS3**: Custom properties, animations, and glassmorphism styling.
*   **Vanilla JavaScript**: State management, collision detection, and logic.
*   **Web Audio API**: Dynamically synthesizes in-game sound effects.

### Features

*   **Customizable Name**: Input screen to personalize the end-game message.
*   **Dynamic Difficulty**: The spawn rate of obstacles increases the longer you survive.
*   **Combo System**: Catching consecutive items builds a score multiplier.
*   **Responsive**: Designed to scale and support both mouse and touch inputs.

### Project Structure

```text
mothers-day-run/
├── index.html   # Main document and UI overlays
├── style.css    # Layout, gradients, and UI styling
├── script.js    # Game engine and audio synthesis
└── README.md    # Project documentation
```

### How to Run

1. Clone or download the repository.
2. Open `index.html` in any modern web browser.
3. No server or build process is required.
